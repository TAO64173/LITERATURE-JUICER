## What the App Does

**Literature Juicer** is a FastAPI web app that:

1. **Takes PDF uploads** — drag-and-drop, multi-file, up to 20 files at 10MB each
2. **Extracts text** via PyMuPDF (front 3 + back 3 pages, skip references)
3. **Analyzes with DeepSeek LLM** — extracts 20 fields per paper (author, year, method, innovation, limitations, etc.)
4. **Exports Excel** — one formatted `.xlsx` file with all papers in a research matrix

It has a card-code (卡密) quota system, SSE progress streaming, and an autonomous orchestrator for AI-driven development.
